## 场景

就是在命令行里的打印一个二维码，
部署完之后可以直接用手机扫，
很方便。

https://github.com/gtanner/qrcode-terminal

### Install
```js
$ npm install qrcode-terminal
or
$ yarn add qrcode-terminal
```

### Usage
```js
qrcode.generate('This will be a QRCode, eh!')
```
