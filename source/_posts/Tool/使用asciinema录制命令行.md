## 场景
有时候我们需要分享一个操作的过程，
这个过程用文字来表示的话会有些抽象，
而录像的成本又太高了，
这个时候`asciinema`会帮到你。

## 安装
```bash
$ brew install asciinema
```
## 使用
用下面的命令开始录制，
结束的时候用`Ctrl-D`。
```bash
$ asciinema rec
~ Asciicast recording started.
~ Hit Ctrl-D or type "exit" to finish.
```

结束后会给你一个地址，
如`https://asciinema.org/a/6j0amorx7tm217lhh1zme1ayc`，
上去可以看到实际的效果，
选择自己的合适的分享方式就可以了。
